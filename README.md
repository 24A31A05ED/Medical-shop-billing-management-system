# MediCore - Medical Shop Billing Management System

## 1. Project Description

MediCore is a web-based Medical Shop Billing Management System developed using HTML, CSS, and JavaScript. It helps in managing medicine inventory, generating bills, and tracking sales efficiently through an interactive dashboard.

## 2. Problem Statement

Manual management of medical shop billing and inventory is time-consuming and prone to errors. This project aims to automate billing, organize medicine records, and provide real-time insights into sales and stock using a user-friendly interface.

## 3. Features of the Project

* User Authentication (Login & Signup UI)
* Dashboard with real-time statistics
* Inventory Management (Add, View, Search medicines)
* Category-based filtering (Tablets, Syrups, Injections, Capsules)
* Billing System with automatic calculation (Tax & Discount)
* Receipt generation and printing
* Sales Analytics and transaction history
* Responsive and modern UI using Tailwind CSS

## 6. Installation / Setup

1. Clone the repository:
   git clone https://github.com/24A31A05ED/Medical-shop-billing-management-system/new/main
2. Navigate to the project folder:
   cd medical-shop-billing

3. Open the project:
   Open index.html in any web browser

(No additional setup required as this is a frontend-based project)

## 7. Usage

* Open the application in a browser
* Register or login to access the system
* Add medicines in the inventory
* Create bills by selecting medicines and quantities
* View total amount with tax and discount
* Generate and print receipt
* Analyze sales data in the dashboard
